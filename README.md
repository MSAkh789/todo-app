# 🚀 Todo App

A modern and responsive Todo application built with **HTML**, **CSS**, and **Vanilla JavaScript**.

**🔗 Live Demo:**
https://msakh789.github.io/todo-app/

</div>

---

## ✨ Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* ❌ Delete tasks
* 💾 Automatically saves tasks using Local Storage
* 📱 Responsive design
* ⚡ Fast and lightweight
* 🌐 Runs directly in the browser
* 🚫 No frameworks required

---

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (ES6)

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/MSAkh789/todo-app.git
```

Open the project folder and run:

```text
index.html
```

No installation required.
